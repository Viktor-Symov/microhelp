=========================================
=== Справка по lsof.
=== инструмент с открытым исходным кодом, для получения информации о файлах используемых OC в текущее время.
=========================================

Утилита lsof по умолчанию выводит следующую информацию о каждом файле:
 - COMMAND — название процесса или команды, которая использует файл;
 - PID — идентификатор процесса;
 - USER — пользователь, которому принадлежит процесс;
 - FD — дескриптор файла
    cwd - Текущий рабочий каталог
    rtd - Корневой каталог
    txt - Текст программы (код и данные)
    mem - Файл памяти
    r - Режим доступа для чтения
    w - Режим доступа для записи
    u - Режим чтения и записи
 - TYPE — тип файла или объекта файловой системы (например, REG — обычный файл, DIR — каталог, и другие);
    DIR - Каталог
    REG - Обычный файл
    CHR - Специальный символьный файл.
    FIFO - First In First Out
 - DEVICE — мажорный и минорный номера устройства, на котором находится файл;
 - SIZE/OFF — размер файла или значение смещения в нем;
 - NODE — номер индексного дескриптора (inode);
 - NAME — абсолютный путь к файлу.

1. Показать информацию о файлах, которые используются процессами конкретного пользователя:
    $ lsof -u user

2. Показать информацию о файлах, которые прослушивающих какой-либо порт:
    $ lsof -i

3. Показать информацию о файлах пользователя, использующих сетевые соединения:
    $ lsof -i -u user

4. Показать информацию о файлах пользователя (кроме определенного), использующих сетевые соединения:
    $ lsof -i -u ^root

5. Показать информацию о процессах, которые работают на определенном порту:
    $ lsof -i TCP:22
    $ lsof -i TCP:22-1028

6. Показать информацию о процессах, которые используют IPV4 / IPV6:
    $ lsof -i 4
    $ lsof -i 6

7. Показать информацию о файлах, которые открыты конкретным процессом:
    $ lsof -p 1

8. Показать информацию о идентификаторах процессов, запущенных конкретным пользователем:
    $ lsof -t -u user

9.  Завершить все процессы пользователя:
    $ kill -9 $(lsof -t -u user)
