
Справка по PKG. Только FreeBSD.

1. Получить отчет по версиям установленных пакетов:
 # pkg version

2. Найти пакет в репозитории pkg:
 # pkg search <pkg-name>

3. Найти и показать в каком разделе репозитория находится пакет:
 # pkg search -o p<pkg-name>

4. Установить скомпилированный пакет из репозитория:
 # pkg install <pkg-name>

5. Установка пакета из локального источника:
 # pkg add <pkg-name>
 # pkg add <protocol>://<path>/<pkg-name>

6. Показать информацию об установленном пакете:
Если пакет не установлен, будет сообщено "pkg: No package(s) matching mc"
 # pkg info <pkg-name>

7. Поиск по установленным пакетам
 # pkg info | grep ftp

8. Запросить информацию с удаленного репозитория:
 # pkg rquery <pkg-name>

9. Удалить установленный пакет:
 # pkg remove <pkg-name>
 или
 # pkg delete <pkg-name>

10. Если есть необходимость запретить изменение или удаление пакета, то можно его заблокировать:
 # pkg lock <pkg-name>

11. Разблокировать пакет:
 # pkg unlock <pkg-name>

12. Показать куда установлена команда или исполняемый файл пакета:
# pkg which /path/to/executable

13. Скачать пакетиз репозитория, но не устанавливать его:
 # pkg fetch <pkg-name>
 # ls /var/cache/pkg/ | grep <pkg-name>

14. Проверить список имеющиеся уязвимостей и багов в пакете:
 # pkg audit -F
 # pkg audit <pkg-name>

15. Обновить локальную базу пакетов из репозиотрия:
 # pkg update

16. Обновить все установленные пакеты из репозиотрия:
 # pkg upgrade

17. Показать суммарную информацию по установленным пакетам и пакетам находящихся в репозитории:
 # pkg stats

18.Удалить зависимости:
 # pkg autoremove

19. Очистка локального кеша, данных данных о удаленных пакетах, которые out-of-date и пакеты которые больше не сопровождаются:#
 # pkg clean -a -n  
 # pkg clean -a -y
